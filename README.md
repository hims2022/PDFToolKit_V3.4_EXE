1. 功能栏
   在主窗口上端有设置功能栏，其中有四个详细选项，分别为“文件(F)”“PDF工具(T)”“历史文件”和“帮助(H)”，四个选项除“历史文件”下又有详细分支，分别为“文件(F)”-“打开文件”或“退出”；“PDF工具(T)”-“PDF合并”，“PDF分割”，“页面提取”，“Office转PDF”，“PDF转Office”，“PDF加密”，“批量打印”；“帮助(H)”-“关于”，“赞赏作者”，可用这些功能按钮快捷跳转到相应功能。其中，点击“历史文件”，用户可以查看历史转换的文件，“历史转换”所显示的文件不随程序的关闭而消除。
2. 标题与副标题
   标题与副标题分别为“PDF工具箱V3.4”“多功能PDF文档处理工具 - 支持加密PDF和密码检测”
3. 菜单栏
   菜单栏功能选项分别为“合并”，“分割”，“提取”，“转换”，“反向转换”，“保护”，“打印”，分别点击可切换至各自功能的窗口
4. 功能主题组件
   4.1 “合并”
   使用该功能，单击“添加文件”以添加需合并的PDF文件，如需移除已上传的文件，则需单击显示在组件上方“待合并文件”区域中需删除的文件名称，再单击“移除”以移除文件。如需调整文件顺序，则需单击显示在组件上方“待合并文件”区域中需删除的文件名称，单击“上移”或“下移”以调整，调整完成后，当前显示的文件顺序则为输出合并文件中的顺序。最后，在“输出文件名”中输入文件名，点击“开始合并”并在新弹出的“浏览”窗口选择输出目录，即可完成合并。
   4.2 “分割”
   使用该功能，单击“浏览”以添加需分割的PDF文件，分割选项中分别有“每页一个文件”，“固定页数分割”，“按页码范围”三个选项可供用户选择，若选择后两种分割模式，则需按实例填入数值以分割。单击“开始分割”并在新弹出的“浏览”窗口选择输出目录，即可完成分割。
   4.3 “提取”
   与（II）“分割”相似，故不过多介绍。
   4.4 “转换”
   使用该功能，单击“添加文件”以添加需转换的Office文件（支持*.doc，*.docx，*.ppt，*.pptx，*.xls，*.xlsx），如需移除已上传的文件，则需单击显示在组件上方“待转换文件”区域中需删除的文件名称，再单击“移除”以移除文件。单击“开始转换”并在新弹出的“浏览”窗口选择输出目录，即可完成转换。
   4.5 “反向转换”
   使用该功能，单击“添加文件”以添加需反向转换的PDF文件，如需移除已上传的文件，则需单击显示在组件上方“待转换PDF文件”区域中需删除的文件名称，再单击“移除”以移除文件。如需调整转换顺序，则需单击显示在组件上方“待转换PDF文件”区域中需删除的文件名称，单击“上移”或“下移”以调整。上传文件后，组件中会显示当前上传文件是否加密，如加密，则在转换过程中须输入密码以解锁，如多个文件对应密码相同，则无需多次输入。在“转换选项”区域中，“输出格式”包括“Word”“Excel”“Powerpoint”，每个选项有其对应的下拉选项，请依照程序实际和自身需求选择。单击“开始转换”并在新弹出的“浏览”窗口选择输出目录，即可完成转换。
   4.6 “保护”
   使用该功能，单击“浏览”以添加需添加保护的PDF文件，“选择文件”下方有“检查加密状态”按钮，可检查当前文件是否加密，如已加密，则在接下来的添加保护过程中，须输入密码以移除当前保护，以添加新的保护。本功能也提供“移除密码保护”功能。在使用“测试密码”功能时，每个用户每次有三次测试密码的机会，若连续三次输入密码错误，则需退出重新测试。
   4.7 “打印”
   使用该功能，单击“添加文件”以添加需打印的PDF文件，如需移除已上传的文件，则需单击显示在组件上方“待打印文件”区域中需删除的文件名称，再单击“移除”以移除文件。如需调整文件顺序，则需单击显示在组件上方“待打印文件”区域中需删除的文件名称，单击“上移”或“下移”以调整，调整完成后，当前显示的文件顺序则为打印完成后文件中的顺序。在“打印设置”中，用户可以自行选择打印机，并选择“合并为一个文件打印”，该复选框默认为勾选状态。单击“批量打印”以开始打印。
